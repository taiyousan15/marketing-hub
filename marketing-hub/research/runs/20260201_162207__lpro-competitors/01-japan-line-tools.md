# Japan Market - LINE Marketing Tools

## Overview
The Japanese LINE marketing tools market is dominated by several key players that extend LINE Official Account functionality. None directly replicate LPRO's automatic user distribution and BAN redirect features, but all offer multi-account management capabilities.

---

## 1. Lステップ (L-STEP)

**Website:** https://linestep.jp/
**Developer:** 株式会社Maneql

### Description
LINE公式アカウント専用のマーケティングオートメーション（MA）ツール。LINEで成約率を高める機能を備えた拡張ツール。

### Key Features
- シナリオ配信・シナリオ分岐
- セグメント配信
- 行動スコアリング
- 流入経路分析
- クロス分析 (Proプラン)
- スタッフ権限設定 (Proプラン)
- 複数店舗の一括管理（各店舗専用QRコード発行）

### Multi-Account Management
- **1 LINE公式アカウントにつき1ツール必要**
- 複数運用する場合は、LINE公式アカウント数に応じたツール数を契約
- フリープランは1社1アカウントまで

### BAN Prevention
- APIを活用したツールからメッセージ送信で凍結リスク軽減
- 友だち一人ひとりに最適化されたメッセージ配信で通報リスク低減

### Pricing
| プラン | 月額料金 | メッセージ数 |
|--------|----------|-------------|
| スタートプラン | 低価格帯 | 5,000通/月 |
| プロプラン | 中価格帯 | 50,000通/月 |
| 大量送信プラン | 従量課金 | 無制限 |

- 初期費用なし
- 1ヶ月の無料利用期間あり（通常プラン）
- 解約は3カ月後から可能

### API Availability
- LINE Messaging API連携

### Sources
- [Lステップ公式](https://linestep.jp/lp/01/)
- [料金プラン](https://linestep.jp/lp/01/plan.html)

## LINE BAN Prevention Best Practices (Japan Market)

### 凍結される主な原因
1. 利用規約違反
   - アダルト、出会い系
   - ネットビジネス、ギャンブル
   - マルチ商法
   - 「稼ぐ系」（投資、アフィリエイト）
2. ユーザーからの通報
3. アフィリエイト広告掲載（規約第18条で禁止）

### 推奨対策
1. **API拡張ツールの活用**
   - LINE公式アカウントから直接送信より凍結リスクが低い
   - Lステップ、エルメ等の利用

2. **複数アカウント運用によるリスク分散**
   - メインアカウントとサブアカウントを用意
   - メイン登録者にサブへの登録を促す

3. **他チャネルへの誘導**
   - メルマガ、SNSへの登録促進
   - LINEが凍結されても顧客リストを失わない

4. **AIツールによる規約チェック**
   - Claude、ChatGPT等でメッセージ案の規約違反チェック
