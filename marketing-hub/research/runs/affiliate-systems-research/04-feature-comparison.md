### Detailed Platform Analysis

#### MyASP (マイスピー)

**Strengths:**
- Most comprehensive affiliate features
- 2-tier commission support
- VIP rate customization per affiliate
- Strong API for integrations
- Mature platform with large community
- Flexible commission triggers (signup, trial, payment)

**Weaknesses:**
- Higher learning curve
- No free tier
- Interface can feel dated
- No native LINE-only option

**Best For:**
- Serious marketers needing full control
- Businesses with complex affiliate structures
- Those needing API integrations

**Affiliate Feature Rating:** ★★★★★ (5/5)

---

#### UTAGE (ウタゲ)

**Strengths:**
- Modern, clean interface
- All-in-one marketing suite
- 30% recurring affiliate program (platform's own)
- AI-powered features
- Competitive pricing for features offered

**Weaknesses:**
- No 2-tier commissions
- Affiliate only on Standard plan
- Smaller community than MyASP
- Limited API access

**Best For:**
- Newcomers wanting all-in-one solution
- Course creators and coaches
- Those prioritizing UX over flexibility

**Affiliate Feature Rating:** ★★★☆☆ (3/5)

---

#### エキスパ (EXPA)

**Strengths:**
- Triple delivery (Email/LINE/SMS)
- Public and private affiliate campaigns
- Integrated with larger affiliate network
- Some features free to use

**Weaknesses:**
- Focus on paid products only
- No 2-tier support
- Less affiliate flexibility than MyASP
- Mixed reviews on reliability

**Best For:**
- Multi-channel marketers
- Those with existing EXPA affiliate network
- SMS marketing needs

**Affiliate Feature Rating:** ★★★☆☆ (3/5)

---

#### Proline Free (プロラインフリー)

**Strengths:**
- Free tier with ASP functionality
- Unique "everyone is an affiliate" model
- Built-in fraud detection
- Fastest payout (5 days)
- Multi-tier (MLM) support

**Weaknesses:**
- LINE-only focus
- No email marketing
- Limited to LINE friend acquisition
- Simpler feature set

**Best For:**
- LINE-focused businesses
- Those wanting free ASP option
- Viral growth strategies

**Affiliate Feature Rating:** ★★★★☆ (4/5)

---

#### LIGET (リゲット)

**Strengths:**
- Dedicated LINE opt-in affiliate platform
- Integration with Lステップ
- Focused feature set

**Weaknesses:**
- Limited to LINE only
- No email support
- Requires other tools for full marketing

**Best For:**
- Lステップ users
- Pure LINE affiliate campaigns
- Simple opt-in programs

**Affiliate Feature Rating:** ★★★☆☆ (3/5)

## 6. Scalability Analysis

### Small Business (< 100 affiliates)

**Recommended Japan:** MyASP Standard or Proline Free
**Recommended Global:** Refersion or Systeme.io

**Key Needs:**
- Easy setup
- Low cost
- Basic tracking
- Simple reporting

---

### Mid-Market (100-1,000 affiliates)

**Recommended Japan:** MyASP Standard with API customization
**Recommended Global:** Post Affiliate Pro or Tapfiliate

**Key Needs:**
- Multi-tier support
- Fraud detection
- API access
- Advanced reporting

---

### Enterprise (1,000+ affiliates)

**Recommended Japan:** Custom solution or MyASP Mach + integration

**Key Needs:**
- White-label
- ML fraud detection
- SOC 2 compliance
- Custom integrations
- High volume handling

## 8. Key Takeaways

1. **Japan is unique** - MyASP dominates for comprehensive affiliate features
2. **LINE is essential** - All Japan platforms now support LINE
3. **2-tier is standard in Japan** - MyASP and Proline offer this
4. **Global platforms are more integrated** - Better webhooks, APIs
5. **Fraud prevention varies** - Enterprise tools (TUNE) lead
6. **Pricing scales with features** - Free options exist but limited
7. **Integration matters** - API access differentiates platforms
